### Installatioin
#### Clone the repository
```bash
https://github.com/SaveDim/python-project-lvl2
```
Use make (Makefile):


```bash
make install
make build
```


### Usage

to start enter:
```bash
gendiff
```
for help use: 
```bash
gendiff -h
```
for example:
```bash
gendiff path_to_first_file path_to_second_file
```
### You can also choose format: 
stylish

plain

json

for example: 
```bash
gendiff --format json path_to_first_file path_to_second_file
```
